# Engineering Graduate Job Market Analyzer

## Project Overview

This project analyzes hiring trends for Data Analyst and Business Analyst roles using Power BI. The dashboard provides insights into job demand, salary benchmarks, hiring velocity, work models, cloud platform adoption, and skill requirements across industries.

---

## Business Problem

Engineering graduates often struggle to identify:

* Which skills are most demanded
* Which industries offer the highest salaries
* Preferred work models (Onsite, Hybrid, Remote)
* Locations with the highest hiring activity
* Emerging tools and technologies

This dashboard transforms job market data into actionable career insights.

---

## Key Insights

* Mumbai emerged as one of the strongest hiring locations.
* SQL, Power BI, Tableau, and Excel remain highly demanded skills.
* Hybrid and Onsite roles dominate hiring trends.
* Cloud platforms such as AWS and Azure are increasingly requested.
* Analytics and consulting companies offer competitive salary ranges.
